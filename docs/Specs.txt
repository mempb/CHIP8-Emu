CHIP-8 Specifications:

- Memory:
    - 4 KB RAM
    - Layout:
        - 0xFFF - top of RAM
        - 0x200 - ROM loads here, where PC starts
        - 0x000 - 0x1FF - reserved (interpreter, font data lives here)

- Display:
    - 64 x 32 pixels, monochrome

- Font:
    - 16 characters (0-F), each 5 bytes tall
    - Stored at 0x050-0x09F

- PC, points to current instruction in memory

- 1 16-bit index register "I", points at locations in memory

- Stack for 16-bit addresses, used to call subroutines/functions and return from them

- 8-bit delay timer, decremented at 60 Hz

- 8-bit sound timer, functions like delay timer, bit gives off beeping sound while != 0

- 16 8-bit general-purpose variable registers V0-VF
    - VF also flag reg

Instructions:
- Implement these first
- 00E0 (clear screen)
- 1NNN (jump)
- 6XNN (set register vx)
- 7XNN (add value to register vx)
- ANNN (set indec register I) 
- DXYN (display / draw)
